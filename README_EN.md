# Cookie Clicker 2.0

A modern Cookie Clicker game built with React + TypeScript + Vite.

## Features

- Game logic implemented with React Hooks and TypeScript
- Responsive UI design for different devices
- Persistent game data storage
- Upgrade and achievement systems

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: CSS Modules

## Project Structure

```
src/
├── assets/            # Static assets
│   ├── cookie_icon.png
│   └── upgrade_icon.png
├── components/        # Components
│   └── CookieClicker.tsx
├── types/             # Type definitions
│   └── game.ts
├── App.tsx            # Main component
└── main.tsx           # Entry file
```

## Run the Project

1. Install dependencies
```bash
npm install
```

2. Start dev server
```bash
npm run dev
```

3. Build production version
```bash
npm run build
```
